# 说明
[Hugo]是由[Go]语言实现的静态网站生成器。简单、易用、高效、易扩展、快速部署。

# 常用命令
    # 创建站点
    hugo new mysite
    # 创建博客
    hugo new posts/mypost.md
    # 启动hugo
    hugo server -D
    # 生成文档，生成的文件在public目录
    hugo 
    # 推送文档到服务器


[Hugo]: https://github.com/gohugoio/hugo "Hugo Github"
[Go]: https://github.com/golang/go "Golang Github"
